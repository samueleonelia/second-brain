# Smart note-taking process
* Write fleeting notes
* Edit the fleeting notes to define a more permanent concept.
* Check the author bibliographical details.
* Discard the fleeting notes.
* Archive the reading notes with a tag.
* Link notes together.
* Use notes for content creation
