# Dofollow vs nofollow links
Google sees dofollow links as a transfer of trust from a website to another.
To distinguish them check the HTML <a href> tag. If the tag contains *rel=”nofollow”*, it is a nofollow link. 
If it doesn’t have it, or contains “dofollow”, it’s a dofollow link. 
Aim for 80% dofollow.
