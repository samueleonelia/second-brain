# Web design best practices
[[Web design best practices/Web design indicazioni]]
1. **Minimalism**
   1. Only the absolute necessary
2. **Contrast**
   1. Highlight the important elements in an unmistakable way.
   2. Only 3 colors: black (dark), white and a color of contrast for both.
3. **Order**
   1. Position everything to the left, except what has to stand out.
   2. Leave white spaces
   3. Create an ordered navigation

1. **Minimalismo** 
   1. Solo l’indispensabile. 
   2. Solo 3 colori.
2. **Contrasto** 
   1. Evidenzia gli elementi più importanti in modo inconfondibile. 
   2. Usa bianco, nero e un colore di contrasto su entrambi.
3. **Ordine**
   1. Posiziona tutto a sinistra, tranne le eccezioni.
   2. Lascia spazi bianchi
   3. Crea una navigazione ordinata
