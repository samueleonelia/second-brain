# How to make a better content
*”Better”* in this case means mostly **different from competitors**.
* **“Better” quality** -> If everybody else writes short guides, go into more details. If everybody writes long tutorial go super brief and to the point.
* **“Better” voice** -> Add more casual personality if everybody is super formal. Or viceversa.
* **“Better” organization** -> If everybody else is full of huge unorganized paragraph, offer more readable articles. Or curate better the subheads.
* **“Better” positioned** -> Offer a radically different view on the topic.
* **“Better” for the audience** -> Go more specific or broader 
* **“Better” user experience** -> If others push hard for the sale, give the whole knowledge for free. 
* **“Better” data** -> focus on one original data, or give more than the competitors
* **“Better” quotes**
* **“Better” insight**
* **“Better” stories**
* **“Better” clarity**
* **“Better” examples** -> Use insider examples, or on the contrary pick from mainstream figures, or from completely unrelated categories.
* **”Better” introduction** -> Especially for lists, if nobody takes advantage of this section.
