# How to write the main points of an article
You can use one of this templates:
* 1/2/5/3/1
* 1/3/1
* 1/1/1+
* Main Point/1, 2 or 3
* Bolded Statements
* Short/Long/Long/Short
* Repetition
* First, Second, etc.
* Bold 1/2  Sentences
