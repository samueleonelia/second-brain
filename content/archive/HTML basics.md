# HTML basics
*Width=“Xpx” define the width of an image, but Colt says is not to use. WHY?*
* **<div>** groups together different elements (and they will occupy the minimum amount of space possible)
* **<**, **>** and **&** are reserved character, instead of them use entity codes
* When we use a **<form>** we submit information into a webpage
  * the *action* attribute specify where to send the data
  * the *method* attribute specifies which HTTP method to use
