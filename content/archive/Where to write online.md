# Where to write online
Especially at first don’t write on a private blog.
Ideally write on:
1. a social platform,
2. that has traction or a huge userbase,
3. it’s working to become profitable (but without a robust advertising model in place, yet).

The third point is an ideal not always possible. So don’t get hanged up on that. Give priority to the first two elements.
