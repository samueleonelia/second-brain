# Proposal sequence
1. Initial contact
2. Conceptual agreement on call
   1. Situation analysis
   2. Goal definition
   3. Value assessment
   4. Anticipate you will send the proposal within 24 hours
   5. Set already the next call to see *“what option best suits you”*
3. Send the proposal within 24 hours
4. Have the call to 
   1. see what option best suits the client, 
   2. face his uncertainty,
   3. or learn what he didn’t like in case of refusal.
