# Best note-taking apps
1. Bear
   * Pro: it’s the most beautiful writing experience
   * Con: only for Apple users
2. Notion
   * Pro: versatile
   * Con: too versatile
3. Simplenote
   * Pro: 100% free
   * Con: no nested tags
4. Obsidian
   * Pro: graph view
   * Con: need a learning phase
5. One Note
   * Pro: advanced note taking function (web clipper, OCR)
   * Con: ugly
